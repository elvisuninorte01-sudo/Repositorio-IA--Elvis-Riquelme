# Proyecto RAG con LangChain y Mistral AI

## Descripción

Proyecto de Inteligencia Artificial enfocado en la creación de un sistema RAG (Retrieval-Augmented Generation) utilizando un dataset empresarial previamente normalizado.

El sistema permite realizar consultas inteligentes sobre el dataset mediante recuperación semántica y modelos de lenguaje.

---

## Objetivo del proyecto

Crear un modelo capaz de acceder y consultar información del dataset utilizando:

- Pandas
- LangChain
- Mistral AI
- Embeddings
- FAISS
- RAG

---

## Tecnologías utilizadas

- Python
- Pandas
- LangChain
- Mistral AI
- FAISS
- Sentence Transformers
- Google Colab

---

## Flujo del proyecto

Dataset CSV
↓
Pandas
↓
Documentos de texto
↓
Embeddings
↓
FAISS
↓
Retriever
↓
Prompt contextualizado
↓
Mistral AI
↓
RAG
↓
Respuesta final

---

## Funcionalidades implementadas

- Limpieza y normalización del dataset
- Conversión del dataset a documentos de texto
- Creación de embeddings
- Almacenamiento vectorial con FAISS
- Recuperación semántica de información
- Integración con Mistral AI
- Implementación de sistema RAG con LangChain
